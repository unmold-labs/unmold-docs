# Setting Up Your User Account

To publish and consume modules in **Unmold**, you first need to create a user account.

## Sign Up

Go to the [signup page](https://unmold.dev/signup) and you can create an account using one of the following methods:

* **Email and password**
* **GitHub (OAuth)**

## Option 1 — Email and Password

Provide the following information:

* **Email**
  Used to sign in and receive account notifications.

* **Username**
  Your personal namespace for modules (for example: `username/module-name`).
  Requirements:

  * Lowercase only
  * Can include hyphens and numbers

* **Password**
  Must be at least 10 characters and include a mix of letters, numbers, and symbols.
  Using a password manager is strongly recommended.

## Option 2 — GitHub Sign-In

Sign up using your GitHub account:

* Your GitHub username will be used as your Unmold username
* You’ll authorize Unmold to access basic profile information

## Terms and Privacy

Before creating your account, review our [Terms of Service](https://unmold.dev/terms) and [Privacy Policy](https://unmold.dev/privacy).

## Verify Your Account

After creating the account, you will receive a verification email in the provided email address. You must verify the address before using the registry service.
