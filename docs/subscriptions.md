# Subscriptions

Unmold offers two plans: **Free** and **Pro**, designed to support individuals and professional users.

## Overview

Subscription plans control usage limits, including how many modules you can publish and manage in the registry.

## Free Plan

The Free plan is ideal for individuals and small-scale usage.

* Full access to core registry functionality
* Publish 1 private module with unlimited versions
* Suitable for local development and experimentation

## Pro Plan

The Pro plan is designed for professional users.

* All features from the Free plan
* Publish 50 modules with unlimited versions
* Make modules public or private
