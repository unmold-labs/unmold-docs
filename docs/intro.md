# Introduction

Unmold is a developer-friend private registry service for [OpenTofu](https://opentofu.org) and [Terraform](https://developer.hashicorp.com/terraform) modules. It is particularly designed to support development-time plication:
* Simple module publication and management with CLI
* Flexible module publication with unlimited version names and number
* Integration with modern CI/CD tools such as GitHub Actions

## Use Cases

* Publish preview modules on PR push to test in other projects
